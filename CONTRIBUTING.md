# Contributing to discord-autoshard

Thank you for your interest in contributing! Here are some guidelines.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/uxoxufe/discord-autoshard.git`
3. Create a branch: `git checkout -b feature/my-feature`
4. Install dependencies: `npm install`

## Development

### Running Tests
```bash
npm test
npm run test:watch
npm run test:coverage
```

### Linting
```bash
npm run lint
npm run lint:fix
```

### Code Style

- Use 2 spaces for indentation
- Use single quotes for strings
- Add semicolons
- Follow ESLint rules

## Pull Requests

1. Update tests if needed
2. Update documentation
3. Follow commit message conventions
4. Ensure all tests pass
5. Update CHANGELOG.md

## Commit Messages

Format: `type(scope): message`

Types:
- `feat`: New feature
- `fix`: Bug fix
- `docs`: Documentation
- `style`: Formatting
- `refactor`: Code restructuring
- `test`: Tests
- `chore`: Maintenance

Example: `feat(sharding): add dynamic scaling support`

## Questions?

Open an issue or join our Discord server.
